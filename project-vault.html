Vault Escape

2D runner built with Java (Swing) and tested with JUnit. Multiple levels and simple state management; built for clarity and maintainability.

Project Overview

Vault Escape is a 2D runner game, designed to emphasize clarity, modularity, and strong gameplay feel. The project combines handcrafted assets, responsive controls, and dynamic level generation to create a challenging but accessible experience.

The game was built using Java Swing and JavaFX, with robust unit testing implemented through JUnit. The level creation system was initially designed to support complex procedural mechanics, but later refined into a simpler and more maintainable approach for efficient gameplay tuning.

Overall, Vault Escape reflects an iterative development process that was based ob best OOP principles, focused on balancing creativity and simplicity — combining technical rigor with a polished, minimalist design philosophy.
